题目与答案

已知二次函数 \( f(x) = ax^2 + bx + c \) 的图像经过点 \( (1, 3) \)、\( (2, 0) \) 和 \( (-1, -5) \)，求 \( a \)、\( b \)、\( c \) 的值。

正确答案：

将三个点代入函数得到方程组：

\[ \begin{cases} a + b + c = 3 \\ 4a + 2b + c = 0 \\ a - b + c = -5 \end{cases} \]

解得：\( a = -1 \)，\( b = 4 \)，\( c = 0 \)，即 \( f(x) = -x^2 + 4x \)。

错因分析
  • 未正确建立方程组，忽略了二次项系数 \( a \) 的影响
  • 解方程组时计算错误，特别是在消元过程中符号处理不当
  • 没有验证解是否满足所有三个点，导致答案不完整
  • 对二次函数的基本形式理解不深，无法将点的坐标正确代入
知识点总结与拓展

知识点总结：

本题主要考察二次函数的基本形式 \( f(x) = ax^2 + bx + c \) 以及通过已知点确定函数解析式的方法。关键在于理解三个点的坐标对应三个方程，从而建立三元一次方程组并求解。

拓展应用：

  1. 已知二次函数图像的顶点和另一点，求函数解析式
  2. 利用二次函数解决实际问题中的最大值或最小值
  3. 分析二次函数的开口方向、对称轴和零点
  4. 通过图像变换（平移、伸缩、对称）推导新的二次函数

解题技巧：

当已知二次函数图像经过三个点时，直接代入函数式建立方程组是最基本的方法。在解方程组时，应优先消去同一个未知数，简化计算过程。解得系数后，务必将结果代入原方程验证，确保正确性。
